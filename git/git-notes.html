Git

Git is a version control system, it can manage source code history. Git saves changes locally.

Git consists of repositories, branches and commits.

Repository - The location where your code is stored.

Branch - "Folder" within the repository containing commits.

Commits - Differents stages of your code inside a branch.

Commands

$git init

Initialization of a git repository inside a project folder to be managed by git.

$git status

Display the status of monitored/tracked files.

$git add .

Add files in the project to be tracked. '.' for all files.

$git commit -m "..."

Save current code as a version in the branch, "..." will be the name of the commit.

$git branch

View all branches.

$git log

View all commits. The latest commit is at the top. HEAD marks the currently active commit.

$git checkout (id of commit)

View a commit. This commit becomes the HEAD but is no longer inside its branch.

$git master

Return to master branch.

$git reset --hard (id of commit)

Go to a commit and delete all commits after that.

$git checkout -b (branch-name)

Create a new branch with all previous commits.

$git checkout (branch-name)

Switch to specified branch.

$git merge (branch-name)

Merge branches.

$git branch -D (branch-name)

Delete a branch.

GitHub

GitHub is a hosting and collaboration provider. It is a web based git repository.

Allows the storage of local git repositories in the cloud, turning them into remote repositories.

Commands

$git remote add origin (link of repository)

Establish a connection between a local and remote repository. origin is the name, it can be changed.

$git remote

Shows the name of the remote repository.

$git remote -v

Shows the name and link of the remote repository.

$git push -u origin master

Pushes branches to the remote repository. '-u origin master' is necessary for the first push. Pushing requires a username and password.

$git branch -r

Shows the connection between the remote and the local repository.

$git clone (link of repository)

Creates a copy of a remote repository on the local machine.

$git push

Pushes changes to the remote repository.

$git fetch

Fetches changes from the remote repository.

$git checkout origin/master

Looking at changes after fetching.

$git merge origin/master

Merges the local and the remote repository.

$git pull

Fetches from the remote repository and merges with the local repository.

$git remote rm (name of repository)

Deletes a repository.
